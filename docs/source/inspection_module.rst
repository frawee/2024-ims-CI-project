Inspection Module
==================

The Inspection Module in the Brewing Package provides a test that lets Prof. Snape examine your potion.

Constants
---------

.. automodule:: brewing.brew_potions
   :members:

   This module contains a test function for your potion.


Usage
-----

You can use the Ingredients Module to access the magical ingredients required for brewing potions. Here's an example:

.. code-block:: python

    from brewing import inspection

    # Accessing ingredients
    

For more information on the inspection, refer to the documentation provided above.
