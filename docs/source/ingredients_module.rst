Ingredients Module
==================

The Ingredients Module in the Brewing Package provides a collection of magical ingredients for brewing potions.

Constants
---------

.. automodule:: brewing.ingredients
   :members:

   This module contains constants representing various magical ingredients used in potion-making.


Usage
-----

You can use the Ingredients Module to access the magical ingredients required for brewing potions. Here's an example:

.. code-block:: python

    from brewing import Ingredients

    # Accessing ingredients
    print("Available Ingredients:")
    print("Snake Skin:", Ingredients.snake_skin)
    print("Fish Eyes:", Ingredients.fish_eyes)
    print("Unicorn Hair:", Ingredients.unicorn_hair)
    print("Tea Leaves:", Ingredients.tea_leaves)

For more information on the Ingredients Module and available ingredients, refer to the documentation provided above.
